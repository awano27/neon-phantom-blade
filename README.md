# Neon Phantom Blade

Phaser 3 で作ったスマホ対応のネオン忍者アクションゲームです。

## ローカル実行

`index.html` を静的サーバーで開いてください。

例:

```powershell
python -m http.server 8123
```

そのあと `http://127.0.0.1:8123/` を開きます。

## GitHub Pages

このリポジトリには GitHub Actions で GitHub Pages へ公開する設定が入っています。

1. GitHub の `Settings > Pages` を開く
2. `Build and deployment` を `GitHub Actions` にする
3. `main` に push する

公開後は GitHub Pages のURLからスマホで遊べます。
